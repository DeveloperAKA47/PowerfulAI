路径参数是 URL 路径中的动态部分，使用花括号 {} 声明

路由顺序
当多个路由可能匹配同一个 URL 时，定义的顺序决定了匹配结果。
在 FastAPI（以及大多数 Web 框架）中，路由按代码中定义的顺序从上到下依次匹配。一旦某个路由匹配成功，就会执行对应的处理函数，后续路由不再检查。

from fastapi import Path
Path 常用参数：
参数	类型	说明	示例
default	Any	路径参数没有默认值概念，通常用 ... 表示必需	Path(...)
ge	float	数值 ≥ 该值	ge=1
le	float	数值 ≤ 该值	le=1000
gt	float	数值 > 该值	gt=0
lt	float	数值 < 该值	lt=100
min_length	int	字符串最小长度	min_length=3
max_length	int	字符串最大长度	max_length=20
regex	str	正则表达式校验	regex=r"^[A-Za-z]+$"
description	str	在 API 文档中显示的描述	description="书籍的唯一ID"
title	str	参数标题（文档用）	title="Book ID"
注意：路径参数通常不需要设置 default，因为路径必须包含路径参数。一般写作 Path(...) 。
