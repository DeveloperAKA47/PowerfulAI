## 参考提示

- 读取 CSV：`pd.read_csv('sales.csv')`
- 查看缺失值：`df.isnull().sum()`
- 日期转换：`pd.to_datetime()`
- 分组聚合：`df.groupby().agg()`
- 透视表：`pd.pivot_table()`
- 保存 Excel：`df.to_excel('文件名.xlsx', sheet_name='名称')`，需安装 `openpyxl` 或 `xlsxwriter`

---

```python
import numpy as np
import pandas as pd
df = pd.read_csv('sales.csv', encoding='utf-8')
#1. 读取数据
print(df.head(5))#查看前五行
print(df.info())#查看基本信息（类型、缺失）
print(df.describe()) # 查看统计信息（均值、最值）
print(df.shape) #查看结构（行数, 列数）
print(f"----------------")
#2.数据清洗
#检查各列是否存在缺失值，若有则处理（自行决定填充或删除，并说明理由）
print(df.isna().sum())  #查询是否存在缺失值
#理由：经代码查验，无缺失值，无需处理
#将 日期 列转换为 datetime 类型？？？？？？
df = df.dropna() #若有可以按照此代码删除含缺失的行
df['日期'] = pd.to_datetime(df['日期'])
#添加新列 **销售额**，计算公式为：`销量 * 单价`。
df['销售额'] = df['销量'] * df['单价']
#将 **产品** 列转换为 `category` 类型，节省内存（可选）。？？？？
#3. 数据筛选与排序
#筛选出 销售额大于 1500 的所有记录。
result1 = df[df['销售额'] > 1500]
print(result1)
#按 日期 升序排列整个 DataFrame。
df = df.sort_values(by='日期', ascending=True)
#4. 分组统计（基础）
#- 按 **产品** 分组，计算：- 总销量 - 总销售额 - 平均单价 - 记录条数（即该产品出现的次数）
result2 = df.groupby('产品').agg(
    总销量=('销量', 'sum'),
    总销售额=('销售额', 'sum'),
    平均单价=('单价', 'mean'),
    记录条数=('产品', 'count')
).reset_index()
print(result2)
# - 将结果按总销售额降序排列。
result2 = result2.sort_values(by='总销售额', ascending=False)
print(result2)
#6. 日期维度分析
# 提取 **月份**（假设所有日期在同一年），按月份分组，统计每月的总销售额和总销量。
# 按月份绘制简单的折线图（可选，鼓励使用 matplotlib 或 seaborn 进行可视化）。
df['月份'] = df['日期'].dt.month
months = df.groupby('月份').agg({'销售额': 'sum', '销量': 'sum'}).reset_index()
### 7. 保存结果到 Excel 文件
#将以下内容分别写入同一个 Excel 文件 **`sales_report.xlsx`** 的不同 sheet 中：
#- Sheet1: 原始清洗后的完整数据（包括新增的“销售额”列）
#- Sheet2: 按产品分组统计结果（任务4的结果）
#- Sheet3: 按地区×产品的透视表（任务5的结果）
#- Sheet4: 按月统计结果（任务6的结果）
#确保 Excel 文件格式规范（如列宽自适应、数字格式等），至少要求保存成功
with pd.ExcelWriter('sales_report.xlsx') as writer:
 df.to_excel(writer,sheet_name='原始清洗后的完整数据')
 result2.to_excel(writer,sheet_name='按产品分组统计结果')
 months.to_excel(writer,sheet_name='按月统计结果')
```
