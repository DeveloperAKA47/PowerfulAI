### 第8题：装饰器执行顺序

不运行代码，写出下面程序的完整输出顺序：

```python
def deco_a(func):
    def wrapper(*args, **kwargs):
        print("A before")
        result = func(*args, **kwargs)
        print("A after")
        return result
    return wrapper

def deco_b(func):
    def wrapper(*args, **kwargs):
        print("B before")
        result = func(*args, **kwargs)
        print("B after")
        return result
    return wrapper

@deco_a
@deco_b
def add(a, b):
    print("执行 add")
    return a + b

print("结果:", add(3, 5))
```

然后将装饰器语法糖展开成普通赋值语句，即写出与下面代码等价的形式：

```python
@deco_a
@deco_b

    ...
```

最后说明多个装饰器“从下往上包装、从上往下进入执行”的含义。
