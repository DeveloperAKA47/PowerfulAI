### 第6题：自定义迭代器

编写一个自定义迭代器类 `EvenNumbers`，满足以下要求：

```python
for number in EvenNumbers(3, 12):
    print(number)
```

输出：

```text
4
6
8
10
```
class EvenNumbers:
    def __init__(self, start, end):
        self.current = start + 1
        self.end = end
    def __iter__(self):
        return self   
    def __next__(self):
        if self.current >= self.end:
            raise StopIteration
        value = self.current
        self.current += 2
        return value
for number in EvenNumbers(3, 12):
    print(number)
具体要求：构造方法接收 `start` 和 `end`；只返回 `[start, end)` 范围内的偶数；实现 `__iter__()` 和 `__next__()`；遍历结束时正确抛出 `StopIteration`；同一个迭代器遍历完成后，再次遍历不能从头开始。

### 第8题：装饰器执行顺序

不运行代码，写出下面程序的完整输出顺序：

```python
def deco_a(func):
    def wrapper(*args, **kwargs):
        print("A before")
        result = func(*args, **kwargs)
        print("A after")
        return result
    return wrapper

def deco_b(func):
    def wrapper(*args, **kwargs):
        print("B before")
        result = func(*args, **kwargs)
        print("B after")
        return result
    return wrapper

@deco_a
@deco_b
def add(a, b):
    print("执行 add")
    return a + b

print("结果:", add(3, 5))
```
#进入 deco_a 的 wrapper：打印 A before
#调用 func，实际是 deco_b 的 wrapper，打印 B before
#调用原函数 add：打印 执行 add，返回 8
#回到 deco_b wrapper：打印 B after，把 8 返回给外层
#回到 deco_a wrapper：打印 A after，返回 8
#执行最后的 print：结果: 8
#然后将装饰器语法糖展开成普通赋值语句，即写出与下面代码等价的形式：

```python
@deco_a
@deco_b
def add(a, b):
    ...
```

    print("执行 add")
    return a + b
