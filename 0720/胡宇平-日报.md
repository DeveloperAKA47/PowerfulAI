深拷贝创建一个全新的复合对象，并递归地复制原对象内部所有层级的子对象。最终产生一个与原对象完全独立、不共享任何可变元素的副本。

不可变元素，浅拷贝和深拷贝效果一致

通过闭包的方式定义一个装饰器

```py
def decorator_with_args(arg1, arg2):
    def real_decorator(func):
        def wrapper(*args, **kwargs):
            # 使用 arg1, arg2
            return func(*args, **kwargs)
        return wrapper
    return real_decorator
```
