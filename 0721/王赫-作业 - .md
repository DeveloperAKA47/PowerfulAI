
### 文本单词统计

编写函数：

```python
def word_statistics(text: str) -> dict[str, int]:
    ...
```

函数接收一段英文文本，统计每个单词出现的次数。

例如：

```python
text = "Python is good, Python is easy. Good code is important!"
```

要求：

1. 不区分大小写，`Python` 和 `python` 视为同一个单词。
2. 忽略以下标点符号：

```text
, . ! ? : ;
```

3. 使用字典保存统计结果。
4. 返回结果中只保留出现次数大于等于2次的单词。
5. 空字符串或只包含空格时，返回空字典。
6. 不允许使用第三方模块。

预期结果：

```python
{
    "python": 2,
    "is": 3,
    "good": 2
}
```

可以使用的基础方法包括：

```python
str.lower()
str.replace()
str.split()
dict.get()
```

```python
答案：
import re
def word_statistics(text: str) -> dict[str, int]:
    #转成小写不区分大小写
    text = text.lower()
    #标点替换
    text = text.replace(","," ")
    text = text.replace("."," ")
    text = text.replace("!"," ")
    text = text.replace("?"," ")
    text = text.replace(":"," ")
    text = text.replace(";"," ")
    #内容的字母替换为列表，word应该算是字典吧？
    words = re.findall(r"\b\w+\b", text)
    #根据words字典里的内容来统计次数
    counts = {}
    for w in words:
        counts[w] = counts.get(w, 0) + 1
    #返回结果中只保留出现次数大于等于2次的单词
    result = {}
    for words,count in counts.items():
        if count >= 2:
          result[words] = count
        if words == { } or words == " ":
          result[words] = " "
    return result
print(word_statistics("Python is good, Python is easy. Good code is important!"))
```
