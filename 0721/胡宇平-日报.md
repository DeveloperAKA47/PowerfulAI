## NumPy
```py
import numpy as np
```
NumPy 的核心数据结构叫 ndarray N-dimensional array，即 N 维数组

| 对比项   | Python 列表                  | NumPy 数组 (ndarray)         |
| :------- | :--------------------------- | :--------------------------- |
| 元素类型 | 可以混合不同类型             | 所有元素类型相同             |
| 内存占用 | 较大（每个元素是独立的对象） | 小（连续内存块）             |
| 运算效率 | 慢（需要循环逐个处理）       | 快（向量化操作，C 语言实现） |

np.array() 将 Python 列表或元组转换为 ndarray。

np.zeros(shape)	创建全 0 的数组

np.ones(shape)	创建全 1 的数组

np.full(shape, value)	创建指定值的数组

np.eye(N)	创建 N×N 单位矩阵

np.arange(start, stop, step)	返回数组

np.array([1], dtype=np.int32)  创建指定数据类型的数组

arr.astype(np.float32)  转换数据类型

ndarray的属性

shape	形状，即各维度的大小	

ndim	数组的维度数	

size	数组中元素的总个数	

dtype	元素的数据类型	

itemsize	每个元素占用的字节数	

nbytes	数组总字节数（size × itemsize）

```py
scores = np.array([45, 78, 92, 33, 88])
pass_scores = scores[scores >= 60]
```

numpy的向量化是指对数组的所有元素同时执行相同的操作，不需要显式编写循环。NumPy 在底层使用 C 语言循环

广播  自动把小数组“拉伸”成和大数组一样的形状。

广播规则：从后往前对齐维度，缺失或为 1 的维度可以扩展

reshape函数  改变数组的形状，不改变数据本身。

矩阵乘法

```py
import numpy as np
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])
C = np.dot(A, B) 
D = A @ B
```
