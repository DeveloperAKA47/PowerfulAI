## 二、作业文件与统一开发规则

`numpy_ai.pyi` 是仅供查看的接口文件，不要直接在 `.pyi` 中编写实现。`.pyi` 是类型声明文件，实际代码应写`.py`文件：


```python
raise NotImplementedError
```

允许使用：

```python
import math
import numpy as np
```

以及Python标准库中的类型、异常和辅助函数。禁止使用：

```text
PyTorch
TensorFlow
JAX
SciPy
scikit-learn
pandas
```

统一要求如下：必须保持 `.pyi` 中的函数名称、参数和返回类型不变；核心数组计算必须使用NumPy向量化，不允许逐元素编写Python循环；输入首先使用 `np.asarray()` 转换；整数数组参与除法、指数和距离运算前应转换成浮点数组；参数不合法时抛出 `ValueError`，数据类型不合法时抛出 `TypeError`，形状不合法时抛出已有的 `ShapeError`；今晚暂时约定 `eps=None` 时使用 `1e-7`，`one_hot()` 的 `dtype=None` 时使用 `np.float32`，以后实现 `RuntimeConfig` 时再统一接入配置系统。

允许成员编写不公开的私有辅助函数以减少重复，例如：

```python
_as_float_array(...)
_validate_reduction(...)
_apply_reduction(...)
```

这些私有函数不是额外作业题，不要求与pyi文件中的框架设计的隐藏参考实现名称完全一致。

## 如何看懂函数接口声明

以 `one_hot()` 为例：

```python
def one_hot(
    indices: ArrayLike,
    num_classes: int,
    *,
    dtype: Any | None = None,
) -> np.ndarray:
    ...
```

接口声明描述的是：**函数叫什么、接收哪些参数、参数应该是什么类型、哪些参数可以省略，以及函数应该返回什么类型**。它规定开发者必须实现什么，但不提供具体实现代码。

### 1. 函数名称

```python
def one_hot(
```

表示声明一个名为 `one_hot` 的函数。开发时函数名不能随意修改，否则其他代码无法按照接口调用它。

### 2. 必填参数

```python
indices: ArrayLike
```

参数名是 `indices`，用于接收类别编号，例如：

```python
[2, 0, 1]
```

冒号后面的 `ArrayLike` 是类型标注，表示可以接收能够转换成NumPy数组的数据，例如列表、元组或 `np.ndarray`。

```python
num_classes: int
```

参数名是 `num_classes`，要求传入整数，表示类别总数。例如共有3个类别时：

```python
num_classes = 3
```

前两个参数没有默认值，因此调用函数时必须提供：

```python
one_hot([2, 0, 1], 3)
```

### 3. 星号 `*`

```python
*
```

星号表示：**星号后面的参数只能使用参数名传入，不能按位置传入**。

因此下面是正确调用：

```python
one_hot([2, 0, 1], 3, dtype=np.float32)
```

下面是不允许的：

```python
one_hot([2, 0, 1], 3, np.float32)
```

这样设计可以让调用代码更清晰，看到 `np.float32` 时能明确知道它表示 `dtype`。

### 4. 可选参数和默认值

```python
dtype: Any | None = None
```

参数名是 `dtype`，用于指定输出数组的数据类型。

```python
Any | None
```

表示该参数可以接收某种数据类型对象，也可以接收 `None`。

```python
= None
```

表示它有默认值，因此调用时可以省略：

```python
one_hot([2, 0, 1], 3)
```

也可以主动指定：

```python
one_hot([2, 0, 1], 3, dtype=np.float64)
```

开发者需要在实现中决定：当 `dtype is None` 时使用什么默认类型。本次作业约定默认使用：

```python
np.float32
```

### 5. 返回值类型

```python
-> np.ndarray
```

表示函数执行完成后，应该返回一个NumPy数组。

例如：

```python
result = one_hot([2, 0, 1], 3)
```

`result` 应该是：

```python
np.ndarray
```

而不是普通Python列表。

### 6. 省略号 `...`

```python
...
```

在 `.pyi` 文件中，省略号表示这里只声明接口，不提供函数实现。

成员真正开发时，应在 `.py` 文件中写出函数体：

```python
def one_hot(
    indices: ArrayLike,
    num_classes: int,
    *,
    dtype: Any | None = None,
) -> np.ndarray:
    labels = np.asarray(indices)

    # 参数检查和具体计算
    ...

    return result
```

### 7. 类型标注不会自动完成检查

下面的标注：

```python
num_classes: int
```

只是告诉开发者和代码检查工具，这个参数应该是整数。Python通常不会因为写了类型标注就自动拒绝错误数据。

因此，必要的类型和数值检查仍然需要自己编写：

```python
if not isinstance(num_classes, int):
    raise TypeError("num_classes must be an integer")

if num_classes <= 0:
    raise ValueError("num_classes must be positive")
```

### 8. 阅读接口时的固定顺序

看到一个接口声明时，按照以下顺序阅读：

```text
函数名
→ 必填参数
→ 可选参数
→ 参数类型
→ 默认值
→ 是否只能按参数名传入
→ 返回值类型
```

例如：

```python
def normalize(
    value: ArrayLike,
    *,
    axis: AxisLike = -1,
    eps: float | None = None,
) -> np.ndarray:
    ...
```

可以读成：

```text
函数名是 normalize。

必须传入 value，它可以是列表、元组或NumPy数组。

axis 和 eps 位于星号后面，因此只能使用参数名传入。

axis 默认是 -1，表示沿最后一个维度计算。

eps 默认是 None，实现时使用约定的默认极小值。

函数最终必须返回 np.ndarray。
```

合法调用：

```python
normalize([3, 4])

normalize(
    [[3, 4], [5, 12]],
    axis=1,
    eps=1e-7,
)
```

开发时应严格保持接口中的函数名、参数名、参数顺序、默认值以及返回结果含义不变。可以在函数内部增加局部变量、参数检查和私有辅助函数，但不能自行改变公开接口。

## 三、正式作业题

### 第1题：实现 `one_hot()`，建议30分钟

接口：

```python
def one_hot(
    indices: ArrayLike,
    num_classes: int,
    *,
    dtype: Any | None = None,
) -> np.ndarray:
    ...
```

它的作用是把类别编号转换成独热编码。例如类别总数为4时，类别编号2转换为：

```text
[0, 0, 1, 0]
```

输入：

```python
indices = np.array([2, 0, 1])
num_classes = 3
```

输出形状应为：

```text
(3, 3)
```

输出内容应为：

```text
[[0, 0, 1],
 [1, 0, 0],
 [0, 1, 0]]
```

实现思路是：先用 `np.asarray()` 将输入变成数组；检查数据类型是否为整数；检查 `num_classes > 0`；检查所有类别编号都位于 `[0, num_classes)`；创建形状为 `indices.shape + (num_classes,)` 的全零数组；将结果临时重塑为二维数组；使用行索引和类别索引，把对应位置设为1。

需要了解的NumPy函数和属性：

```python
np.asarray
np.issubdtype
np.integer
np.any
np.zeros
np.arange
array.shape
array.size
array.reshape
```

这里不需要复杂数学，只需要理解：每个类别编号对应单位向量中的一个位置。

正常测试：

```python
result = one_hot([2, 0, 1], 3)
assert result.shape == (3, 3)
```

异常测试应包括非整数类别、负数类别、类别编号超出范围以及 `num_classes <= 0`。

### 第2题：实现 `l2_norm()` 和 `normalize()`，建议30分钟

接口：

```python
def l2_norm(
    value: ArrayLike,
    *,
    axis: AxisLike = None,
    keepdims: bool = False,
) -> np.ndarray:
    ...
```

```python
def normalize(
    value: ArrayLike,
    *,
    axis: AxisLike = -1,
    eps: float | None = None,
) -> np.ndarray:
    ...
```

L2范数可以理解为向量的“长度”。对于：

```text
[3, 4]
```

长度为：

```text
sqrt(3² + 4²) = 5
```

数学公式：

```text
L2(x) = sqrt(sum(x²))
```

`l2_norm()` 的实现顺序是：转换为浮点数组；逐元素平方；沿指定 `axis` 求和；对结果开平方；正确传递 `keepdims`。

需要使用：

```python
np.asarray
astype
np.sum
np.sqrt
```

也允许直接使用：

```python
np.linalg.norm
```

但推荐用 `sqrt(sum(x * x))`，因为更能体现实现原理。

`normalize()` 是把向量除以自己的长度：

```text
normalize(x) = x / L2(x)
```

例如：

```text
[3, 4] → [0.6, 0.8]
```

关键点是计算范数时必须保留维度：

```python
keepdims=True
```

这样分母才能通过广播与原数组相除。零向量的长度为0，不能直接作为分母，所以使用：

```python
np.maximum(norm, eps)
```

将分母最低限制为 `eps`。零向量归一化后仍然是零向量。

测试：

```python
assert np.allclose(l2_norm([3, 4]), 5.0)
assert np.allclose(normalize([3, 4]), [0.6, 0.8])
assert np.allclose(normalize([0, 0]), [0.0, 0.0])
```

### 第3题：实现 `cosine_similarity()`，建议30分钟

接口：

```python
def cosine_similarity(
    x: ArrayLike,
    y: ArrayLike,
    *,
    axis: int = -1,
    eps: float | None = None,
) -> np.ndarray:
    ...
```

余弦相似度用于判断两个向量的方向有多接近，常用于文本向量、Embedding、推荐系统和语义检索。

数学公式：

```text
cosine(x, y) =
sum(x * y) / (L2(x) * L2(y))
```

只需要知道三个结论：方向相同结果接近1，方向相反结果接近-1，互相垂直结果接近0。

实现思路：将两个输入转换成浮点数组；使用 `np.broadcast_arrays()` 检查并统一广播形状；计算 `x * y` 后沿 `axis` 求和，得到点积；分别计算两个向量的L2范数；两个范数相乘作为分母；使用 `np.maximum(denominator, eps)` 防止除零；返回点积除以分母。

需要使用：

```python
np.broadcast_arrays
np.sum
np.sqrt
np.maximum
```

或者复用已经完成的：

```python
l2_norm
```

测试：

```python
assert np.allclose(cosine_similarity([1, 0], [1, 0]), 1.0)
assert np.allclose(cosine_similarity([1, 0], [-1, 0]), -1.0)
assert np.allclose(cosine_similarity([1, 0], [0, 1]), 0.0)
```

### 第4题：实现 `pairwise_squared_distance()`，建议35分钟

接口：

```python
def pairwise_squared_distance(
    x: ArrayLike,
    y: ArrayLike,
) -> np.ndarray:
    ...
```

输入必须是两个二维矩阵：

```text
x.shape = (m, d)
y.shape = (n, d)
```

其中 `m` 和 `n` 是两组样本的数量，`d` 是每个样本的特征数量。输出形状为：

```text
(m, n)
```

输出的第 `i` 行第 `j` 列表示：

```text
x中的第i个向量与y中的第j个向量之间的平方距离
```

不能编写双重Python循环，必须使用矩阵运算。

需要知道的数学公式只有一个：

```text
||x - y||² = ||x||² + ||y||² - 2(x · y)
```

矩阵实现过程：

```text
x平方后按行求和 → 形状(m, 1)
y平方后按行求和再转置 → 形状(1, n)
x @ y.T → 形状(m, n)
```

利用广播计算：

```text
x平方和 + y平方和 - 2 * 点积矩阵
```

浮点误差可能使理论上的0变成很小的负数，例如 `-1e-15`，最终应使用：

```python
np.maximum(result, 0.0)
```

需要使用：

```python
np.asarray
array.ndim
array.shape
np.sum
array.T
@
np.maximum
```

测试：

```python
x = np.array([[0, 0], [1, 0]])
y = np.array([[0, 0], [2, 0]])

expected = np.array([
    [0, 4],
    [1, 1],
])

assert np.allclose(pairwise_squared_distance(x, y), expected)
```

还要测试：输入不是二维数组、两个矩阵特征维度不同。

这个函数在隐藏参考实现中同样采用“两个平方和加广播，再减去两倍矩阵乘积”的向量化方案。

### 第6题：实现数值稳定的 `sigmoid()`，建议25分钟

接口：

```python
def sigmoid(value: ArrayLike) -> np.ndarray:
    ...
```

普通数学公式：

```text
sigmoid(x) = 1 / (1 + exp(-x))
```

它把任意实数映射到0和1之间，并且：

```text
sigmoid(0) = 0.5
```

但不能对所有元素直接使用原公式。假设 `x = -1000`，计算 `exp(-x)` 就会变成 `exp(1000)`，可能产生溢出警告或无穷大。

因此采用分段等价公式：

```text
x >= 0:
    1 / (1 + exp(-x))

x < 0:
    exp(x) / (1 + exp(x))
```

实现思路：创建与输入相同形状的空结果数组；创建 `x >= 0` 的布尔掩码；分别处理正数区域和负数区域；写入结果。

需要使用：

```python
np.asarray
astype
np.empty_like
np.exp
布尔索引
```

测试必须包含极端值：

```python
result = sigmoid(np.array([-1000.0, 0.0, 1000.0]))

assert np.all(np.isfinite(result))
assert np.allclose(result[1], 0.5)
assert result[0] < 1e-10
assert result[2] > 1.0 - 1e-10
```

隐藏实现也使用正负区域分别计算的方式避免指数溢出。

### 第7题：实现数值稳定的 `softmax()`，建议30分钟

接口：

```python
def softmax(
    value: ArrayLike,
    *,
    axis: int = -1,
) -> np.ndarray:
    ...
```

Softmax把一组任意分数转换成一组概率形式的数值：每个结果大于等于0，并且沿指定轴的总和为1。

数学公式：

```text
softmax(x_i) = exp(x_i) / sum(exp(x))
```

不能直接对很大的值求指数。例如：

```text
[1000, 1001, 1002]
```

直接执行 `exp(1002)` 会溢出。

Softmax有一个重要性质：所有元素同时减去相同的数，最终结果不变。因此先减去该轴的最大值：

```text
shifted = x - max(x)
```

然后计算：

```text
exp_shifted = exp(shifted)
result = exp_shifted / sum(exp_shifted)
```

求最大值和求和时都必须使用：

```python
keepdims=True
```

否则分母形状可能无法正确广播。

需要使用：

```python
np.asarray
np.max
np.exp
np.sum
```

测试：

```python
x = np.array([
    [1.0, 2.0, 3.0],
    [1000.0, 1001.0, 1002.0],
])

result = softmax(x, axis=1)

assert result.shape == x.shape
assert np.all(np.isfinite(result))
assert np.allclose(result.sum(axis=1), [1.0, 1.0])
assert np.allclose(result[0], result[1])
```

标量没有可进行Softmax的轴，因此零维数组应抛出 `ShapeError`。隐藏实现同样先沿指定轴减去最大值，再执行指数和归一化。

### 第8题：实现 `mse_loss()` 和 `mae_loss()`，建议30分钟

接口：

```python
def mse_loss(
    prediction: ArrayLike,
    target: ArrayLike,
    *,
    reduction: Reduction = "mean",
) -> np.ndarray | float:
    ...
```

```python
def mae_loss(
    prediction: ArrayLike,
    target: ArrayLike,
    *,
    reduction: Reduction = "mean",
) -> np.ndarray | float:
    ...
```

MSE是均方误差：

```text
error = prediction - target
loss = error²
```

MAE是平均绝对误差：

```text
error = prediction - target
loss = abs(error)
```

例如：

```python
prediction = np.array([1.0, 3.0])
target = np.array([2.0, 1.0])
```

误差为：

```text
[-1, 2]
```

MSE逐元素结果：

```text
[1, 4]
```

MAE逐元素结果：

```text
[1, 2]
```

`reduction` 必须支持三种模式：

```text
"none"：返回逐元素损失数组
"sum"：返回所有损失之和
"mean"：返回所有损失的平均值
```

其他字符串必须抛出 `ValueError`。

建议编写一个私有辅助函数统一处理：

```text
_validate_reduction
_apply_reduction
```

需要使用：

```python
np.asarray
np.abs
np.sum
np.mean
array.size
```

测试：

```python
prediction = np.array([1.0, 3.0])
target = np.array([2.0, 1.0])

assert np.allclose(
    mse_loss(prediction, target, reduction="none"),
    [1.0, 4.0],
)
assert np.allclose(
    mse_loss(prediction, target, reduction="mean"),
    2.5,
)
assert np.allclose(
    mae_loss(prediction, target, reduction="sum"),
    3.0,
)
assert np.allclose(
    mae_loss(prediction, target, reduction="mean"),
    1.5,
)
```

预测值与目标值允许NumPy正常广播；如果无法广播，应捕获 `ValueError` 并转换成 `ShapeError`。参考模块也把 `"none"`、`"sum"` 和 `"mean"` 集中到统一的私有归约函数处理。

## 六、成员开发时需要掌握的NumPy函数速查

`np.asarray(value)`：把列表、元组或数组统一转换成 `ndarray`，输入已经是数组时通常不会无故复制。

`np.maximum(a, b)`：逐元素取较大值，不等同于只能处理单个对象的Python `max()`。

`np.where(condition, x, y)`：条件为真时选择 `x`，否则选择 `y`，相当于数组版本的条件表达式。

`np.sum(..., axis=..., keepdims=True)`：沿指定维度求和；保留被压缩的维度，方便之后广播。

`np.max(..., axis=..., keepdims=True)`：沿指定维度取最大值，Softmax中用于数值稳定处理。

`np.broadcast_arrays(x, y)`：按照广播规则将两个数组调整成可共同计算的形状，不会真的把所有数据复制多份。

`array.T`：二维数组转置，把 `(n, d)` 变成 `(d, n)`。

`x @ y`：矩阵乘法，不是对应元素相乘。

`np.isfinite()`：判断结果是否不是 `nan`、正无穷或负无穷，适合测试数值稳定性。

`np.allclose(actual, expected)`：浮点数组不能简单依赖 `==`，应允许极小的浮点误差。
