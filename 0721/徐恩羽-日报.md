利用np.array()将 Python 列表或元组转换为 ndarray
dtype
指定数据类型
int32 / int64	整数	np.array([1,2], dtype=np.int32)
float32 / float64	浮点数	np.array([1,2], dtype=np.float64)
bool	布尔值	np.array([True, False])

数组长什么样？—— 查看属性
属性	含义	示例（假设 arr 是 2行3列的数组）
shape	数组的形状，即各维度的大小	(2, 3)
ndim	数组的维度数（轴的数量）	2
size	数组中元素的总个数	6
dtype	元素的数据类型	int64
itemsize	每个元素占用的字节数	8
nbytes	数组总字节数（size × itemsize）	48
